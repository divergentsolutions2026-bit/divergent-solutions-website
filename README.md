# Divergent Solutions AI Digital Agency Website

A modern, responsive portfolio website for Divergent Solutions AI Digital Agency, showcasing AI automation services for real estate companies.

## Features

- **Modern Design**: Clean, professional design with gradient accents
- **Fully Responsive**: Works on all devices from mobile to desktop
- **Interactive Elements**: Smooth animations, hover effects, and form validation
- **Performance Optimized**: Fast loading with minimal dependencies
- **SEO Ready**: Proper semantic HTML structure
- **Accessibility**: Keyboard navigation and screen reader friendly

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern CSS with Flexbox, Grid, and CSS Variables
- **JavaScript**: Vanilla JS for interactivity
- **Font Awesome**: Icon library
- **Google Fonts**: Inter and Poppins fonts

## Project Structure

```
agency-website/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # All styles
├── js/
│   └── script.js      # JavaScript functionality
├── assets/            # Images and icons (placeholder)
└── README.md          # This file
```

## Sections

1. **Navigation**: Fixed header with smooth scrolling
2. **Hero**: Eye-catching introduction with animated stats
3. **Services**: Showcase of AI automation services
4. **Portfolio**: Case studies and work examples
5. **About**: Agency information and team
6. **Contact**: Contact form and information
7. **Footer**: Links and social media

## Customization

### Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --primary: #4361ee;
    --primary-dark: #3a56d4;
    --secondary: #7209b7;
    --accent: #f72585;
    --dark: #1a1a2e;
    --light: #f8f9fa;
}
```

### Content
Update text content in `index.html`:
- Company name and tagline
- Service descriptions
- Portfolio items
- Contact information

### Images
Replace placeholder images in:
- Hero section floating cards
- Portfolio section images
- About section team image

## Deployment

### GitHub Pages
1. Push to GitHub repository
2. Go to Settings → Pages
3. Select branch (usually `main`)
4. Select folder (`/root`)
5. Save and wait for deployment

### Netlify
1. Drag and drop folder to Netlify
2. Or connect GitHub repository
3. Automatic deployment on push

### Vercel
1. Import GitHub repository
2. Automatic deployment
3. Custom domain support
